# 📊 Sales Data Analysis Dashboard (SQL + Power BI)

## 🚀 Overview
Built a complete data analysis project using SQL for data extraction and Power BI for visualization to analyze sales performance, customers, and product insights.

---

## 🛠 Tools Used
- SQL (Data Extraction & Analysis)  
- Power BI (Dashboard & Visualization)  
- Excel (Data Source)  

---

## 📊 Dashboard Highlights
- Total Revenue: 984.81M  
- Sales Quantity: 2M  
- Revenue trend over time  
- Market-wise sales and revenue  
- Top 5 customers  
- Top 5 products  

---

## 📈 Key Analysis
- Delhi NCR contributes highest revenue  
- Sales show fluctuations over time  
- Few customers generate major revenue  
- Top products drive significant sales  

---

## 💡 Key Insights
- Revenue is highly concentrated in top markets  
- Top 5 customers contribute large portion of sales  
- Certain products dominate overall revenue  
- Sales trend shows decline towards 2020  

---

## 🗄 SQL Work
- Used SQL queries to extract and analyze data  
- Performed aggregations (SUM, COUNT)  
- Grouped data by market, customer, and product
